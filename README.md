# 🌿 FloraVision

A modern, responsive plant store landing page built as a **Frontend Assessment** using **React**, **TypeScript**, **Vite**, and **Tailwind CSS**.

## 🚀 Live Demo

**Vercel:** https://flora-vision-sigma-ruddy.vercel.app/

## 🛠️ Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* React Router
* React Icons

## ✨ Features

* Responsive design
* Reusable components
* Modern UI with glassmorphism
* Smooth scrolling navigation
* Clean and maintainable code

## 📦 Getting Started

```bash
git clone https://github.com/Aditya82003/floreVision.git
cd floraVision/frontend
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
